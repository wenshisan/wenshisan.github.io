数学运算和初等函数

Julia 提供了一个贯穿它全部简单数字类型的完整基础算术运算符和按位运算符，也提供了大量便利高效的标准数学函数。

向量化的“点”操作符

 对每个像是 ^ 的二元运算来说都有一个相对应的 “点” 运算 .^ 就是自动被定义为在数组中依次执行元素的 ^ 运算。比如 [1,2,3] ^ 3 是没有定义的，因为没有标准的数学含义去 “三次方” 一个（没有平方）的数组，但是 .^ 3 被定义为按元素（或 “向量地”）产生 [1^3,2^3,3^3] 的结果。类似于像是 ! 或 √ 的一元运算符，也有与之对应的应用到每个元素的操作符 .√ 。

 更加明确的说，a .^ b 被解析为 “点” 调用 (^).(a,b)，这执行了一个广播运算：它可以组合数组和标量、相同大小的数组（依元素次序执行运算）和即使不同形状的数组（比如组合行和列的向量去产生一个矩阵）。而且和所有的向量化 “点调用” 一样，这些 “点运算符” 都有相熔性。比如如果您为数组 A 计算 2 .* A.^2 .+ sin.(A) （或等价的 @. 2A^2 + sin(A)，使用宏 @.），它会在 A 上执行一个独立的循环，为 A 的每个元素都执行 2a^2 + sin(a)。特别是类似 f.(g.(x)) 叠加的点调用也都是相熔的，并且“相邻的”类似于 x .+ 3 .* x.^2 的操作符都和叠加的点调用 (+).(x, (*).(3, (^).(x, 2))) 相等。

 此外 “加点的” 类似于 a .+= b （或 @. a += b） 的升级操作符都被解析为 a .= a .+ b，.= 是一个合适的熔合赋值运算（查看加点语法文档）。

 注意点的语法也同样适用于用户定义的操作符。比如您为克罗内克积（kron）把定义 ⊗(A,B) = kron(A,B) 给了一个方便的中缀语法 A ⊗ B，之后 [A,B] .⊗ [C,D] 将会在没有额外代码的情况下计算 [A⊗C, B⊗D] 。

 和数字字符组合的点运算符可能会有歧义。比如会不清楚 1.+x 的意思是 1. + x 还是 1 .+ x.。因此不允许这种语法，并且这些情况下必须使用在操作符旁加上空格。

注意链式比较的等于行为：

 中间的表达式只会求一次值，而不是两次，因为将会在如果写成 v(1) < v(2) && v(2) <= v(3) 的表达式中求值。然而链式比较中的顺序是没有定义的。这里强烈不推荐在链式比较中使用有其他作用的表达式（比如打印）。如果需要其他作用，需要显式地使用短路 && 运算符（查看短路求值）。

基本初等函数

 Julia 提供了大批的数学函数和运算符。这些数学运算定义在广泛的合理规范的数值种类上，包括整数、浮点数、有理数和复数，任何这种规范可以合理的地方。

 而且这些函数可以使用点语法 f.(A) 以“向量化”的形式应用到数组和其他集合中，例如 sin.(A) 将会计算数组 A 中每一个元素的正弦。

运算符优先级和关联性

Julia 按照从高到低的优先级来应用下列运算的顺序和关联性：

分类|运算符|关联性
语法|. followed by ::|左
幂运算|^|右
一元|+ - √|右[1]
按位提升|<< >> >>>|左
分数|//|左
乘积|* / % & \ ÷|左[2]
加法|+ - | ⊻|左[2]
语法|: ..|左
语法||>|左
语法|<||右
比较|> < >= <= == === != !== <:|没有关联性
流程控制|&& followed by || followed by ?|右
合并|=>|右
赋值|= += -= *= /= //= \= ^= ÷= %= |= &= ⊻= <<= >>= >>>=|右
[1]
 一元运算符 + 和 - 需要显式地给他们的参数加上括号以避免他们和 ++ 等运算符混淆。其他一元运算符的组合都被解析为右关联，比如 √√-a 解析为 √(√(-a)).

[2]
 操作符 + 、++ 和 * 都没有关联性。a + b + c 被解析为 +(a, b, c) 而不是 +(+(a, b), c)。然而 +(a, b, c, d...) 和 *(a, b, c, d...) 的回退方法都默认为左关联求值。

数字转换

Julia 支持三种不同方式的数字转换，区别于他们对转换准确性的控制。

  • 标记 T(x) 或 convert(T,x) 把 x 转换到类型为 T 的值。

    • 如果 T 是浮点类型，结果可能是正或负无穷的最近可表示值。
    • 如果 T 是整数类型，如果 x 不是可用 T 表示的值会抛出一个 InexactError。

  • x % T 转换整数 x 到适合的 x 以 2^n 为模的整数类型为 T 的值，n 是 T 中的位数。换句话说，二进制的表示将会被截断以进行匹配。

  • 舍入函数将类型 T 作为一个可选参数。例如 round(Int,x) 是 Int(round(x)) 速写法。
